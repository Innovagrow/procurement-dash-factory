### Data Requirements:
- **Historical pricing data** - 12+ months for benchmarking
- **Competitor database** - Track suppliers & wins
- **User profiles** - Store preferences & history
- **Alert rules engine** - Real-time notifications

### APIs Needed:
- OpenAI/Claude for AI assistant
- Email service (SendGrid/AWS SES) for alerts
- Background job queue (Celery + Redis) for processing

### Database Schema:
```sql

    period DATE

    user_id INT,
    tender_id VARCHAR,
    added_date TIMESTAMP

    user_id INT,
    type VARCHAR,
    criteria JSONB,
    active BOOLEAN
);
```
